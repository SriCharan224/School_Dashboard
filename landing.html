What it does

Built for teachers who need answers fast

📊

Subject Performance Bars

Color-coded bars show each student's marks per subject with instant grade indicators (A+ to D).

🔍

Search and Filter

Find any student by name or roll number. Filter the whole class by performance tier in one click.

⚠️

Attention Flags

Students below 60% average or with low attendance are automatically flagged so nothing slips through.

📋

Student Detail View

Click any row to open a full breakdown: attendance status, subject-by-subject bars, and overall average.

📁

JSON Data Source

Data loads from a structured JSON file. Update one file to reflect any changes to the class roster.

📱

Responsive Layout

Works cleanly on desktop and tablet. The table and modal adapt to the screen without breaking.
